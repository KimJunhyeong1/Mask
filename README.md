# 마스크 재고 알림 Android app


**공공데이터포털의 마스크 재고 현황 API와 네이버 지도 API를 이용하여 근처 약국 알려주고 구매가능 한 마스크 재고을 네이버 지도상에 아이콘으로 표시하는 애플리케이션**
